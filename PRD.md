> **버전**: v1.1
> **작성일**: 2026-04-01
> **대상**: 지누켐 전 직원 (~50명)

## 1. 제품 개요

### 1.1 배경
지누켐은 화학/IT 하이브리드 소규모 기업으로, 기존 ERP v3.0(연구관리시스템)을 운영 중이다. 현재 **Slack**(메신저), **Jira**(프로젝트/이슈 관리), **Confluence**(문서/위키), **Google Workspace**(캘린더/메일/드라이브)를 이미 사용하고 있다.

인트라넷은 이러한 기존 도구들과 **중복 구현하지 않고**, 각 도구를 연동하는 **통합 허브(Hub)** 역할을 하며, 기존 SaaS로 대체할 수 없는 한국 고유 기능(전자결재, 근태관리 등)을 자체 구현한다.

### 1.2.1 코러스(KORUS) 벤치마킹 요약

> KORUS = KOrean university Resource United System (국립대학자원관리시스템)
> 교육부 구축, 한국사학진흥재단 운영, 37~39개 국립대학 약 80,000명 사용

**적용할 패턴:**

| KORUS 기능 | 적용 내용 |
|:---|:---|
| 5종 문서함 체계 | 결재대기함/합의함/참조함/품의함/발신함 → 전자결재 모듈에 반영 |
| 포틀릿 대시보드 | 사용자별 위젯 커스터마이징 → 대시보드 개인화 |
| 역할 기반 권한 4단계 | 관리자/경영진/팀장/일반직원 → 메뉴 접근 제어 |
| 예산→집행→정산 흐름 | 연구비 관리 모듈 프로세스에 반영 |
| 외부 시스템 연동 | RCMS/IRIS 개념 → Jira/Confluence 연동으로 대체 |

**반면교사 (KORUS 실패 사례 회피):**

| KORUS 문제점 | 지누켐 회피 방안 |
|:---|:---|
| 사용자 의견 무시한 일방 도입 | 직원 피드백 수집 → 점진적 도입 |
| 느린 속도·잦은 에러 | 가벼운 SPA 구조, iOS 스타일 최적화 |
| 복잡한 서식/검색 | 단순 직관적 UI 유지 |
| 기존 데이터 이전 불가 | 기존 ERP v3.0 데이터 마이그레이션 계획 |
| 학사관리 누락 (불완전 시스템) | 핵심 기능 먼저 완성 후 확장 |

### 1.3 비전
> **"50인 화학/IT 기업에 최적화된, 벤토 그리드 대시보드 기반의 모던 인트라넷"**

### 1.4 핵심 설계 원칙
1. **통합 허브(Hub)** — Slack/Jira/Confluence/Google을 하나의 대시보드에서 조회
2. **자체 구현은 SaaS 불가 영역만** — 전자결재, 근태관리, 게시판, 조직도, MSDS
3. **위젯 기반 대시보드** — 네이버 워크플레이스 스타일 벤토 그리드
4. **세미다크 테마** — 다크 사이드바 + 라이트 콘텐츠 (라이트/다크 모드 전환)
5. **한국식 전자결재** — 기안→검토→합의→승인 워크플로우
6. **화학/IT 하이브리드** — MSDS 관리 + Jira 연동 특화
7. **모바일 반응형** — 접이식 사이드바, 스택형 위젯

## 2. 사용자 정의

### 2.1 사용자 역할

| 역할 | 설명 | 예상 인원 |
|:---|:---|:---|

| **일반 직원 (화학)** | 연구과제 관리, MSDS 조회, 안전교육 관리 | 15~20명 |

| **경영지원** | 인사/급여, 매출/매입, 회계 업무 | 3~5명 |

### 2.2 사용자 시나리오 (핵심)

**시나리오 1: 아침 출근 루틴**
> 직원이 인트라넷 접속 → 대시보드에서 출근 버튼 클릭 → 오늘 결재 대기건 확인 → 공지사항 확인 → 일정 확인

**시나리오 2: 연구비 청구**
> 연구원이 연구비 청구서 작성 → 증빙서류 첨부 → 결재 상신 → 팀장 검토 → 경영진 승인

**시나리오 3: MSDS 긴급 조회**
> 화학 연구원이 대시보드 MSDS 검색 위젯에서 물질명 검색 → 즉시 PDF 열람

### 3.3 전자결재 상세 (M02)

**결재 프로세스 (한국 표준)**
```
기안자 → 검토자(팀장) → 합의자(관련부서장) → 승인자(경영진)
         ↓ 반려                ↓ 반려              ↓ 반려
       기안자에게 반환        기안자에게 반환      기안자에게 반환
```

**문서 양식 종류**
- 일반 기안서 / 휴가 신청서 / 지출 품의서 / 구매 요청서
- 출장 신청서 / 교육 신청서 / 연구비 청구서 / 회의록

**핵심 기능**
- 결재선 템플릿 (자주 쓰는 결재선 저장)
- 대리결재 (부재 시)
- 결재 독촉 알림
- 결재 문서 검색/필터
- 결재 통계 (월별 처리 건수, 평균 처리 시간)

### 3.4 근태관리 상세 (M03)

**52시간 근무제 준수 기능**
- 주간 근무시간 실시간 표시 (진행률 바)
- 40시간 초과 시 경고 알림
- 48시간 초과 시 강력 경고
- 52시간 도달 시 차단 알림
- 월간/분기 근무시간 통계

**출퇴근 관리**
- 원클릭 출근/퇴근
- GPS 또는 IP 기반 위치 확인 (선택)
- 지각/조퇴/외출/반차 처리
- 출퇴근 이력 조회

**연차 관리**
- 잔여 연차 표시
- 연차 사용 내역
- 연차 신청 → 전자결재 연동

### 3.5 화학 부문 특화 기능

| 기능 | 설명 |
|:---|:---|
| **MSDS 관리** | 물질안전보건자료 검색/열람/다운로드, 최신 버전 관리 |
| **안전 공지** | 안전 관련 공지 상단 고정, 긴급 푸시 알림 |
