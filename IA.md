## 4. 화면 흐름도 (User Flow)

### 4.1 로그인 → 대시보드

```
[로그인 페이지]
     │
     ├── 이메일 + 비밀번호 입력
     ├── 자동 로그인 체크
     │
     ▼
[대시보드 — 통합 허브]
     │
     ├─→ [출근 버튼 클릭] → 출퇴근 상태 갱신 (자체)
     ├─→ [결재 대기 클릭] → 전자결재 대기함 (자체)
     ├─→ [공지사항 클릭] → 게시판 > 공지사항 (자체)
     ├─→ [일정 클릭] → Google Calendar (새 탭)
     ├─→ [Slack 멘션 클릭] → Slack 앱/웹 (새 탭)
     ├─→ [Jira 이슈 클릭] → Jira 이슈 페이지 (새 탭)
     └─→ [MSDS 검색] → MSDS 관리 (자체)
```

### 4.2 전자결재 흐름

```
[결재 메뉴 진입]
     │
     ├─→ [새 문서 작성]
     │       │
     │       ├── 양식 선택
     │       ├── 내용 작성
     │       ├── 결재선 설정
     │       ├── 첨부파일
     │       │
     │       ├── [임시저장] → 임시저장함
     │       └── [결재상신] → 결재 진행함
     │                          │
     │                ┌─────────┼─────────┐
     │                ▼         ▼         ▼
     │           [검토자]   [합의자]   [승인자]
     │           승인/반려   승인/반려   승인/반려
     │                │         │         │
     │                └─────────┼─────────┘
     │                          │
     │                    ├── 전체 승인 → 결재 완료함
     │                    └── 반려 → 반려함 → 기안자 수정 → 재상신
     │
     ├─→ [결재 대기함] → 문서 열람 → 승인/반려/보류
     ├─→ [결재 진행함] → 진행 상태 확인
     └─→ [결재 완료함] → 문서 열람/출력
```

### 4.3 근태관리 흐름

```
[근태관리 진입]
     │
     ├─→ [출퇴근 기록]
     │       │
     │       ├── [출근] → 출근 시간 기록 → 상태: 근무중
     │       ├── [외출] → 외출 시간 기록 → 상태: 외출
     │       ├── [복귀] → 복귀 시간 기록 → 상태: 근무중
     │       └── [퇴근] → 퇴근 시간 기록 → 상태: 퇴근
     │
     ├─→ [근무 현황]
     │       ├── 52시간 진행률 바
     │       ├── 주간 근무시간 상세
     │       └── 월간 통계 차트
     │
     ├─→ [연차 관리]
     │       ├── 잔여 연차 확인
     │       ├── [연차 신청] → 전자결재 (휴가 신청서)
     │       └── 사용 내역 조회
     │
     └─→ [근태 신청]
             ├── 지각/조퇴 사유서 → 전자결재
             ├── 휴일근무 신청 → 전자결재
             └── 재택근무 신청 → 전자결재
```

### 4.4 게시판 흐름

```
[게시판 진입]
     │
     ├─→ [게시판 선택] (공지/자유/부서별/익명)
     │       │
     │       ├── 게시글 목록 (검색/필터/정렬)
     │       │       │
     │       │       └── [게시글 클릭]
     │       │               │
     │       │               ├── 본문 열람
     │       │               ├── 첨부파일 다운로드
     │       │               ├── 댓글 작성
     │       │               ├── 좋아요
     │       │               └── 수정/삭제 (작성자)
     │       │
     │       └── [새 글 작성]
     │               ├── 제목/내용 입력
     │               ├── 카테고리 선택
     │               ├── 파일 첨부
     │               ├── 중요 공지 설정 (관리자)
     │               └── 등록
     │
     └─→ [검색] → 전체 게시판 통합 검색
```

```
┌─────┬──────────┬──────────┬──────────┬──────────┬─────┐
│ 🏠  │ 전자결재 ✕│ 근태관리 ✕│ 게시판  ✕│ MSDS   ✕│  ◀▶ │
│메인 │          │          │          │          │     │
└─────┴──────────┴──────────┴──────────┴──────────┴─────┘
```

- 메인(대시보드)은 항상 첫 번째 고정 탭
- 자체 구현 모듈 클릭 시 **인트라넷 내 새 탭** 열림
- 외부 도구(Slack/Jira/Confluence/Google) 클릭 시 **브라우저 새 탭**으로 열림
- 탭 닫기(✕) 가능
- 탭 오버플로 시 좌우 스크롤

## 7. 알림 시스템

### 7.1 알림 유형

| 유형 | 트리거 | 표시 위치 |
|:---|:---|:---|
| 결재 요청 | 새 결재 문서 수신 | 사이드바 배지 + 토스트 + **Slack DM** |
| 결재 승인/반려 | 결재 처리 완료 | 토스트 + **Slack DM** |
| Slack 멘션 | Slack에서 멘션 수신 | 대시보드 Slack 위젯 |
| Jira 이슈 할당 | Jira에서 이슈 할당 | 대시보드 Jira 위젯 |
| 공지사항 등록 | 새 공지 등록 | 토스트 + **Slack 채널** |
| 근태 알림 | 출근 미체크, 퇴근 시간 | 토스트 + 팝업 |
| 52시간 경고 | 근무시간 초과 임박 | 모달 경고 + **Slack DM** |
| 연차 촉진 | 연차 소진 기한 | 토스트 알림 |
| 안전 긴급 | 안전 사고/공지 | 풀스크린 모달 + **Slack 채널** |

### 7.2 알림 표시 방식

```
사이드바 배지:  ✅ 전자결재 [3]    ← 빨간 숫자 배지
토스트 알림:    우하단 슬라이드인 → 5초 후 자동 닫힘
모달 경고:      화면 중앙 모달 (확인 필요)
브라우저 알림:  Web Notification API (선택)
```
